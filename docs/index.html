CARNELIAN

A GUI tool for creating your own workflow


Carnelian is a powerful software, designed to providing you a simple and fast way to create a tasks worflow.


Download

Features

Functions

Organize your tasks into customizable functions

Loops

Create basics or conditionals loops within your functions

Variables

Create and manipulate variables like strings , filenames, booleans...

Customization

Rename or change the color of any of your functions, tasks or triggers

Multi-process

You can run as many process as you can want simultaneously

Error handling

Stop your process or execute a specific function when something weird happend, you choose !

Triggers

Execute your workflow when something specific happend, like a key pressed

Auto updates

Carnelian uses an ultra advanced update system to provide you the latest features at any time!
